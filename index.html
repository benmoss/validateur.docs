Validateur

Validateur is a Clojure validation library inspired by Ruby's ActiveModel

Get Started! » See API Reference » Contribute »

Small

Validateur is small: validators are functions, validation sets are higher-order functions, validation results are returned as values.

Flexible

With Validateur, you can create custom validators the most natural way: by defining functions.
